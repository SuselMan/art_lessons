# 02 — UI создания комнаты

Страница перед входом в холст. Настройки фиксируются при создании — менять нельзя.

---

## Поля формы

### Название комнаты
- Текстовое поле, обязательное
- Макс. 50 символов
- Генерируется случайное по умолчанию ("Blue Canvas", "Studio 12" и т.п.)

### Пароль (опционально)
- Чекбокс "Защитить паролем"
- Поле пароля появляется если включено
- Без пароля — присоединиться по ссылке может кто угодно

### Текстура бумаги
Выбор карточками с визуальным превью:
- **Rough** — зернистая, для классического карандаша
- **Smooth** — средняя зернистость
- **Bristol** — гладкая, для чёткой линии

### Размер холста
Выбор карточками + кастомный вариант:

| Вариант | Размер px | Соотношение | Использование |
|---------|-----------|-------------|---------------|
| A4 | 1240 × 1754 | 1:√2 | Стандартный скетч |
| A3 | 1754 × 2480 | 1:√2 | Крупная работа |
| A2 | 2480 × 3508 | 1:√2 | Академическая постановка |
| Square | 1500 × 1500 | 1:1 | Этюд, портрет |
| 16:9 | 1920 × 1080 | 16:9 | Горизонтальная композиция |
| Custom | — | — | Свои пиксели |

**Custom:** два поля (ширина × высота), ограничение 4096 × 4096 px (лимит WebGL).

**Бесконечный холст** — зарезервирован, показываем карточку с "Скоро" (требует тайловой архитектуры).

---

## Flow

```
create-room.html  →  (submit)  →  /room/:id  (index.html с параметрами)
```

На этапе до сервера: параметры передаются через `localStorage` или URL params.
После добавления сервера: POST /api/rooms → { roomId, joinLink }
