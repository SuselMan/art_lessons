# 05 — Мобильные приложения

---

## Этапы

### 5.1 PWA (первый шаг, без Store)

Работает на iOS и Android через браузер, добавляется на рабочий стол.
Не требует ревью Apple/Google. Сделать параллельно с коллаборацией.

- [ ] `manifest.json` — иконка, название, display: standalone
- [ ] Service Worker — кэш ассетов, работа офлайн (только UI, холст требует сети)
- [ ] Мета-теги для iOS (`apple-mobile-web-app-capable` и др.)
- [ ] Адаптивный UI тулбара под маленькие экраны
- [ ] Тест на реальных устройствах iOS/Android

### 5.2 Capacitor (нативные приложения)

Оборачивает веб-приложение в нативный контейнер. ~95% кода переиспользуется.

**Риски:**
- WebGL в WKWebView (iOS) — работает нормально на iOS 14+
- WebGL в Android WebView — на современных устройствах (Android 7+) ок, на старых — проблемы
- Apple App Store: 30% комиссия с подписок через In-App Purchase. Решение: оплата только через сайт, в приложении не предлагать апгрейд.

- [ ] Настройка Capacitor проекта
- [ ] iOS: `ios/` директория, настройка Xcode
- [ ] Android: `android/` директория, настройка Android Studio
- [ ] Haptic feedback при начале штриха (опционально)
- [ ] Тест WebGL производительности на реальных девайсах
- [ ] Публикация в App Store + Google Play
